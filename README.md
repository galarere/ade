# Cpanel-installer
auto install and config cpanel on your CentOs7 Server

# Need:
* CentOs 7 Linx Server
# run:

install git
```bash
sudo yum install git -y
```
cloning repositories in server
```bash
git clone https://github.com/galarere/ade
```
run Auto Script
```bash
cd ade; bash run.sh
```

**Good Luck.**
